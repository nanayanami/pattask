# 概要
「あれ、この案件は目標に対しどこまで進んでいるのだろう…？」という、ビジネスマンが一度は経験する心理的不安を解決するために開発しました。
 日々のタスクを整理し、目標までの道のりを「見える化」することで、ユーザーを適切なスケジュールへと導きます。
 さらに、チーム内での進捗共有機能も搭載しました。
 よって、「誰が、どこで、止まっているか」がひと目でわかるため、チームはスムーズな連携を手に入れることができます。

## サイトテーマ
「仕事の迷子をゼロにする。タスク進捗可視化ツールPattask」

## テーマを選んだ理由
タスク進捗管理は必ず必要なスキルですが、教科として習うことはありません。ましてや、新しい業務内容に取り組む時に、正しい進捗ができているのかわからなくなります。本アプリで整理し、客観的に捉えなおすこと。また第三者からの確認ができるようにすることで誰１人取り残さない体制を構築できたらいいなと思うようになりました。
旅行は多くの人の記録にとってな体験ですが、そのや情報交換を簡単に選べる場がまだ不足していると感じました。個人のブログやSNSでは旅行情報が埋もれがちですが、このアプリでは特別旅行に特化した情報を集め、手軽に共有・閲覧できる仕組みを提供します。

### 【きっかけ】
月20社の提案に追われ、目標に対するの現在地を見失った経験や、提案活動やマネジメント活動のない人が不安症に陥る現場を打破したいと考えました。
そこで、常にゴールから逆算して課題を構造化し、チームが互いの異変に即座に気づける環境をシステムで構築しようと考えるようになりました。

### 【問題点】
・旅行情報がSNSやブログに分散しており、検索しづらい  
・同じ場所に行った人のリアルな感想や評価が探しにくい  
・旅行の記録が散逸し、後から見返しにくい  

### 【解決策】
・「独学」になりがちなタスク管理を仕組み化
　 仕事の基本でありながら習う機会のない進捗管理を、誰でも正しく行えるよう整理します。
・「迷い」を客観視して、一人で抱え込ませない
　 自分の状況を客観的に捉え直すことで、未知の業務に対する不安や遅れを早期に解消します。

第三者が介入できる「誰も取り残さない」体制へ 周囲が状況を確認し、フォローし合える環境を作ることで、チーム全体の挫折を防ぎます。

・旅行に特化した投稿システムを提供し、場所、感想、写真を一括して共有可能に  
・投稿に基づくおすすめスポットや人気スポットのランキング表示  
・カテゴリ別検索機能により、目的地やテーマ（例：グルメ、自然、アクティビティ）に合わせて情報を簡単に検索  

# ターゲットユーザー
・旅行好きの20〜40代  
・新しい旅行先やおすすめスポットを探している人  
・旅行の記録を簡単に残したい人  

# 主な利用シーン
・旅行前：次の旅行先やスポットを探すとき  
・旅行中：訪れた場所を記録しながら、近くのおすすめを調べるとき  
・旅行後：思い出を整理し、他のユーザーと共有するとき
