# 概要
「あれ、この案件は目標に対しどこまで進んでいるのだろう…？」という、ビジネスマンが一度は経験する心理的不安を解決するために開発しました。
 日々のタスクを整理し、目標までの道のりを「見える化」することで、ユーザーを適切なスケジュールへと導きます。
 さらに、チーム内での進捗共有機能も搭載しました。
 よって、「誰が、どこで、止まっているか」がひと目でわかるため、チームはスムーズな連携を手に入れることができます。

## サイトテーマ
「仕事の迷子をゼロにする。タスク進捗可視化ツールPattask」

## テーマを選んだ理由
タスク進捗管理は必ず必要なスキルですが新しい業務内容に取り組む時に、正しい進捗ができているのかわからなくなります。本アプリで整理し、客観的に捉えなおすこと。また第三者からの確認ができるようにすることで誰１人取り残さない体制を構築できたらいいなと思うようになりました。
旅行は多くの人の記録にとってな体験ですが、そのや情報交換を簡単に選べる場がまだ不足していると感じました。個人のブログやSNSでは旅行情報が埋もれがちですが、このアプリでは特別旅行に特化した情報を集め、手軽に共有・閲覧できる仕組みを提供します。

### 【きっかけ】
1人あたり月20社の提案に追われるなかで、自分自身が目標に対するの現在地を見失った経験や、提案活動やマネジメント活動経験のない人が不安症に陥る状況を経験しました。
そこで、状況を打破するために常にゴールから逆算して課題を構造化し、チームが互いの異変に即座に気づける環境をシステムで構築しようと考えるようになりました。

### 【問題点】
+進捗の「ブラックボックス化」による不安
 知識や解決策が共有されず、各自がゼロから悩む
 孤立による挫折

### 【解決策】
・未知の業務に対する「進捗への不安」を解消
　状況を客観的に整理しなおすことで、自分の進め方が正しいかを冷静に判断できます。
・他者の進捗や解決策から学び、ケースワークができる
　第三者の状況をナレッジとして共有し、チーム全体の対応力を引き上げます。
・チーム全員で完遂を目指す
　周囲が状況を把握しフォローし合える状態を作ることで、誰も取り残さない体制を構築します。

# ターゲットユーザー
・旅行好きの20〜40代  
・新しい旅行先やおすすめスポットを探している人  
・旅行の記録を簡単に残したい人  

# 主な利用シーン
・旅行前：次の旅行先やスポットを探すとき  
・旅行中：訪れた場所を記録しながら、近くのおすすめを調べるとき  
・旅行後：思い出を整理し、他のユーザーと共有するとき
