# 概要
「あれ、この案件は目標に対しどこまで進んでいるのだろう…？」という、ビジネスマンが一度は経験する心理的不安を解決するために開発しました。
 日々のタスクを整理し、目標までの道のりを「見える化」することで、ユーザーは適切なスケジュールを組むことができるようになります。
 さらに、チーム内での進捗共有機能も搭載しました。
 よって、「誰が、どこで、止まっているか」がひと目でわかるため、チームはスムーズな連携を手に入れることができます。

## サイトテーマ
「仕事の迷子をゼロにする。パッとタスクを進捗させるツール、Pattask」

## テーマを選んだ理由
タスク進捗管理は必ず必要なスキルですが新しい業務内容に取り組む時に、正しい進捗ができているのかわからなくなります。本アプリで整理し、客観的に捉えなおしたいと思うようになりました。
また、チームメンバーの案件内容を確認できたり、課題解決策を知ることができるため、ケースワークに繋がります。
更に、第三者からの確認ができるようにすることで誰１人取り残さない体制の構築を目指します。

### 【きっかけ】
私自身が、現在の業務において、1人あたり月20社の提案に追われるなかで、自分自身が目標に対するの現在地を見失った経験や、提案活動やマネジメント活動経験のない人が不安症に陥る状況を経験しました。
そこで、状況を打破するために常にゴールから逆算して課題を構造化し、チームが互いの異変に即座に気づける環境をシステムで構築しようと考えるようになりました。

### 【問題点】
- 進捗の「ブラックボックス化」による不安
- 知識や解決策が共有されず、各自がゼロから悩む
- 孤立による挫折

### 【解決策】
- 未知の業務に対する「進捗への不安」を解消<br>
　状況を客観的に整理しなおすことで、自分の進め方が正しいかを冷静に判断できます。
- 他者の進捗や解決策から学び、ケースワークができる<br>
　第三者の状況をナレッジとして共有し、チーム全体の対応力を引き上げます。
- チーム全員で完遂を目指す<br>
　周囲が状況を把握しフォローし合える状態を作ることで、誰も取り残さない体制を構築します。

# ターゲットユーザー
- マネジメント業務や提案業務を行う社会人
- 課題整理を行いたい人
- チーム制作を行っている人
- 学校行事の実行委員やチームプレゼン等を行う学生
